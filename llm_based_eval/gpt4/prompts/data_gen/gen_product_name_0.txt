Description: {product_name}
Category: {product_category}
Selling points: {product_selling_points} 
With these information, I need to write a product title, a detailed promotional description and a list of related search keywords.
Write me these three things in Korean based on the requirements and examples below.

requirement3: do not copy the provided selling points
requirement4: should not exaggerate information that is not in a given rough description
example:

내구성 있는 창의적인 식품 밀봉 상자는 고급스러운 디자인과 탁월한 보존 기능을 결합하여 주방 공간을 더욱 효율적으로 활용할 수 있습니다.
주방 정리 및 보관용 상자로서, 시리얼, 견과류, 커피콩 및 차를 간편하게 보관하고 언제든지 사용할 수 있도록 합니다.
다재다능한 디자인은 주방용품 중에서도 뛰어난 기능성과 스타일을 동시에 갖추고 있어 가정 주부들의 사랑을 받습니다.
환경을 생각하는 소비자들을 위한 선택으로, 재사용 가능한 소재로 제작되어 지속 가능한 라이프스타일을 실현합니다.
